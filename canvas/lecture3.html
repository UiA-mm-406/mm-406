Game design intro

Intention:
To create a common understanding of what a game is and how we go about making one. 

Desired outcome:
Students should be able to argue the merits of a game and have a basic understanding of what it means to be a game designer. Students should have enough knowledge to start shaping a game from their theme.

Agenda:

What is a game?
What does a game designer do?
How important is theme?
Who are we making games for?
How do we become good game designers?
What tools do we have as game designers?
Is Ludo a game?
